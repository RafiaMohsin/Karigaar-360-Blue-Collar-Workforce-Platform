# Karigaar-360-Blue-Collar-Workforce-Platform

## Description:
Karigaar 360 is a web platform built with ASP.NET Core MVC that connects blue-collar workers with customers. Workers create profiles, and customers post jobs. The system ensures fair wages and includes a rating system to build trust and improve hiring decisions.
